## Installation (Local/Manual)

This extension is **not published on the Chrome Web Store**.  
To use it:

1. Download the source code as a ZIP file or clone the repository.
2. Go to `chrome://extensions/` in your Chrome browser.
3. Enable "Developer Mode" in the top right.
4. Click "Load unpacked" and select the project directory.
5. The extension will appear in your toolbar for use.

**Note:**  
You will need to repeat these steps if you move the folder or clear your Chrome user profile.

# Features and Benefits

- **Offline Functionality**  
  Our extension works smoothly even without an internet connection. All your timers and data are stored locally, allowing uninterrupted use anytime and anywhere.

- **Privacy First**  
  We do not collect or transmit any personal data. All information is securely saved only on your browser, ensuring complete user privacy.

- **Persistent Local Storage**  
  Your timer data and settings are automatically saved using `chrome.storage.local`, so they remain intact even when you close your browser or shut down your computer.

- **Fast and Lightweight**  
  Because all data and functions operate locally without relying on external servers, the extension is highly responsive and consumes minimal resources.
